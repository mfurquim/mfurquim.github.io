Hi, I'm Mateus Furquim, a 22 yer old game developer from Brazil.Troubleshooting Exception: "TypeError: Unable to watch path" If you get following error with a big traceback right after Atom starts: TypeError: Unable to watch path you have to increase number of watched files by inotify. For testing if this is the reason for this error you can issue sudo sysctl fs.inotify.max_user_watches=32768 and restart Atom. If Atom now works fine, you can make this setting permanent: echo 32768 | sudo tee -a /proc/sys/fs/inotify/max_user_watches
